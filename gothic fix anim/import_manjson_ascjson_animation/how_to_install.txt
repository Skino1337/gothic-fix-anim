to install save import_manjson_ascjson_animation folder in "blender addons folder" example: (C:\Program Files\Blender Foundation\Blender 4.0\4.0\scripts\addons\)
goto in blender Edit -> Preferences -> Add-ons -> find "MAN and ASC animation file importer in .json format" -> check on bool to enable -> done!

to open anim:
goto in blender File -> Import -> "Gothic Animation (.MAN.json) (.ASC.json)" -> done!
